## FAQ

**Est-ce que j'ai besoin de savoir coder ?**
Non. Vous importez un fichier dans n8n, vous renseignez votre email SMTP et votre base Notion, et c'est tout. Le guide vous accompagne étape par étape avec des captures d'écran.

**Est-ce que n8n est payant ?**
n8n propose une version cloud gratuite pour démarrer, et une version self-hosted entièrement gratuite si vous hébergez vous-même. Dans la plupart des cas, le pack fonctionne sans aucun coût mensuel supplémentaire.

**Est-ce que je peux modifier les emails ?**
Oui, totalement. Les emails sont en HTML simple — vous pouvez modifier les textes, les couleurs, ajouter votre logo. Le guide explique quelles variables remplacer.
