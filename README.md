# AgencyKit — Workflows automation pour agences

AgencyKit est un ensemble de workflows et d'outils d'automatisation conçus pour les agences : nurturing de leads, séquences email, et orchestration de processus métier, le tout pensé pour gagner du temps et réduire les tâches répétitives.

## Structure

- `lead-nurturing/` — Workflows de suivi et qualification des prospects
